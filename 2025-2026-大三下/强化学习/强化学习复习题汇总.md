## 第一讲 强化学习简介

### 核心概念

**1. 决策和预测有什么不同？请举例说明。**

决策直接改变世界（环境），并需要承担后果（如医生直接给病人下达治疗方案）；预测仅产生信号，不改变环境（如AI告诉医生病人的患病概率，最终由医生决策）。决策改变了未来的发展路径，预测只是辅助别人改变世界。

**2. 强化学习的定义是什么？**

通过从交互中学习来实现目标的计算方法。智能体在某种程度上感知环境的状态，采取行动影响状态，并以随时间推移最大化累积奖励为目标。

**3. 强化学习系统的四大要素是什么？**

- **历史（History）**：观察、行动和奖励的序列 $H_t = O_1, R_1, A_1, \ldots, O_t, R_t$
- **状态（State）**：用于确定接下来会发生的事情的信息，是历史的函数 $S_t = f(H_t)$
- **策略（Policy）**：从状态到行动的映射，分为确定性策略 $a = \pi(s)$ 和随机策略 $\pi(a|s) = P[A_t=a|S_t=s]$
- **价值函数（Value Function）**：对未来累积奖励的预测 $Q^\pi(s,a) = \mathbb{E}_\pi[R_{t+1} + \gamma Q^\pi(s', a') | S_t=s, A_t=a]$
- （另加：**模型 Model**，用于模拟环境行为）

**4. 强化学习的方法分类有哪些？**

- **基于价值（Value-based）**：知道什么是好的，有价值函数，无显式策略
- **基于策略（Policy-based）**：知道怎么行动，有策略，无价值函数
- **Actor-Critic**：同时学习策略和价值函数

**5. 深度强化学习带来的关键变化有哪些？**

将深度神经网络用于价值函数和策略近似，实现端到端学习，但带来了新挑战：高维参数空间、训练不稳定、容易过拟合、需要大量数据和高性能计算（CPU采集经验 + GPU训练网络的平衡）。

**6. 深度强化学习的研究前沿有哪些？**

- 基于模拟模型的强化学习（Model-based RL）
- 目标策动的层次化强化学习（Goal-oriented RL）
- 模仿学习（Imitation Learning）
- 多智能体强化学习（Multi-agent RL）

**7. 强化学习的落地场景有哪些？**

无人驾驶、游戏AI、交通灯调度、网约车派单、组合优化、推荐搜索系统、数据中心节能优化、对话系统、机器人控制、路由选路、工业互联网等。

**8. 什么是利用与探索的均衡（Exploration vs. Exploitation）？**

- **利用（Exploitation）**：执行能获得已知最优收益的决策
- **探索（Exploration）**：尝试更多可能的决策，不一定最优，但可能发现更好的策略
- 二者需要平衡，一直利用会陷入局部最优，一直探索则total regret线性增长

### 多臂老虎机

**9. 多臂老虎机的问题模型是什么？**

- 动作集合：$a_i \in \mathcal{A}$，$i=1,\ldots,K$
- 收益函数分布：$\mathcal{R}(r|a_i) = \mathbb{P}(r|a_i)$
- 目标：最大化累积收益 $\max \sum_{t=1}^T r_t$，其中 $r_t \sim \mathcal{R}(\cdot|a_t)$

**10. 多臂老虎机的算法流程是什么？**

1. 初始化：$Q(a_i) := c_i$，$N(a_i) = 0$
2. 主循环 $t = 1:T$：
   - 利用策略 $\pi$ 选取动作 $a$
   - 获取收益 $r_t = \text{Bandit}(a)$
   - 更新计数器：$N(a) := N(a) + 1$
   - 更新估值：$Q(a) := Q(a) + \frac{1}{N(a)}(r_t - Q(a))$

**11. Regret函数是什么？**

- 最优收益：$Q^* = \max_{a_i \in \mathcal{A}} Q(a_i)$
- 单步Regret：$R(a_i) = Q^* - Q(a_i)$
- **Total Regret**：$\sigma_R = \mathbb{E}_{a \sim \pi} \sum_{t=1}^T R(a_t^i)$
- 目标：$\min \sigma_R \Leftrightarrow \max \mathbb{E}_{a \sim \pi} \sum_{t=1}^T Q(a_t^i)$

**12. 贪心策略 vs. ε-greedy策略的比较？**

| | 贪心策略 | ε-greedy策略 |
|---|---|---|
| 动作选择 | $a^* = \arg\max Q(a_i)$ | 以 $1-\epsilon$ 概率贪心，以 $\epsilon$ 概率随机 |
| Total Regret | 线性增长 $\sigma_R \propto T \cdot (Q(a_i) - Q^*)$ | 线性增长，但增长率更小 |
| 缺点 | 无探索，陷入局部最优 | 常量 $\epsilon$ 仍导致线性增长 |

**13. 什么是乐观初始化（Optimistic Initialization）？**

给 $Q(a_i)$ 一个较高的初始化值，增量式更新 $Q(a_i) := Q(a_i) + \frac{1}{N(a_i)}(r_t - Q(a_i))$。有偏估计，但随着采样增加偏差影响减小，仍可能陷入局部最优。

**14. 什么是UCB上置信界法（Upper Confidence Bounds）？**

$$a = \arg\max_{a \in \mathcal{A}} \hat{Q}(a) + \hat{U}(a)$$

其中 $\hat{U}(a) = \sqrt{-\frac{\log p}{2N_t(a)}}$，$N(a)$ 越大，$U(a)$ 越小（不确定性减小）。收敛性：$\lim_{t \to \infty} \sigma_R \leq 8\log t \sum_{a|\Delta_a > 0} \Delta_a$（对数渐近收敛）。

**15. Thompson Sampling方法是什么？**

根据每个动作成为最优的概率来选择动作。实现：根据当前每个动作 $a$ 的预测价值概率分布 $p(Q(a))$ 来采样到其价值 $Q(a)$，选择价值最大的动作。实验效果接近理论渐近最优下界。

## 第二讲 马尔可夫决策过程

**16. 随机过程和马尔可夫过程有什么不同？**

随机过程研究动态随机现象随时间演变的规律，状态转移依赖历史 $\mathbb{P}[S_{t+1}|S_1,\ldots,S_t]$；马尔可夫过程满足**马尔可夫性质**："The future is independent of the past given the present"，即 $\mathbb{P}[S_{t+1}|S_t] = \mathbb{P}[S_{t+1}|S_1,\ldots,S_t]$，当前状态是未来的充分统计量。

**17. MDP五元组是什么？**

$(S, A, P_{sa}, \gamma, R)$：
- $S$：状态集合
- $A$：动作集合
- $P_{sa}$：状态转移概率 $P_{sa}: S \times A \mapsto \Omega(s)$
- $\gamma \in [0,1]$：折扣因子
- $R: S \times A \mapsto \mathbb{R}$：奖励函数

**18. 累计奖励表达式是什么？**

$$R(s_0, a_0) + \gamma R(s_1, a_1) + \gamma^2 R(s_2, a_2) + \cdots$$

折扣因子 $\gamma$ 使智能体更重视即时奖励（"今天的$1比明天的$1更有价值"）。

**19. 目标、策略和价值函数的定义？**

- **目标**：选择能最大化累积奖励期望的动作 $\mathbb{E}[R(s_0) + \gamma R(s_1) + \gamma^2 R(s_2) + \cdots]$
- **策略**：$\pi(s): S \to A$，在状态 $s$ 下采取动作 $a = \pi(s)$
- **价值函数**：$V^\pi(s) = \mathbb{E}[R(s_0) + \gamma R(s_1) + \cdots | s_0=s, \pi]$

**20. 价值函数的Bellman等式是什么？**

$$V^\pi(s) = R(s) + \gamma \sum_{s' \in S} P_{s\pi(s)}(s') V^\pi(s')$$

最优价值函数：$V^*(s) = R(s) + \max_{a \in A} \gamma \sum_{s' \in S} P_{sa}(s') V^*(s')$

最优策略：$\pi^*(s) = \arg\max_{a \in A} \sum_{s' \in S} P_{sa}(s') V^*(s')$

**21. 价值迭代的过程是什么？**

1. 对每个状态 $s$，初始化 $V(s) = 0$
2. 重复直到收敛：对每个状态更新 $V(s) = R(s) + \max_{a \in A} \gamma \sum_{s' \in S} P_{sa}(s') V(s')$

注：过程中没有明确的策略，初始化价值而非策略。

**22. 策略迭代的过程是什么？**

1. 随机初始化策略 $\pi$
2. 重复直到收敛：
   - a) 令 $V := V^\pi$（策略评估）
   - b) 对每个状态更新 $\pi(s) = \arg\max_{a \in A} \sum_{s' \in S} P_{sa}(s') V(s')$（策略改进）

**23. 价值迭代和策略迭代的对比？**

| | 价值迭代 | 策略迭代 |
|---|---|---|
| 初始化 | 价值函数 $V(s)=0$ | 随机策略 $\pi$ |
| 更新方式 | 贪心更新 | Bellman等式更新价值函数（代价大） |
| 收敛速度 | 较慢（空间大时更实用） | 较快（空间小时） |
| 适用场景 | 大状态空间，无状态转移循环 | 小状态空间 |

**24. 同步和异步迭代有什么区别？**

- **同步迭代**：存储两份价值函数拷贝，用 $V_{old}$ 计算 $V_{new}$，再更新
- **异步迭代**：只存储一份，直接原地更新

**25. 学习模型 & 优化策略的算法流程？**

1. 随机初始化策略 $\pi$
2. 重复直到收敛：
   - a) 在MDP中执行 $\pi$，收集经验数据
   - b) 使用累积经验更新对 $P_{sa}$ 和 $R$ 的估计
   - c) 执行价值迭代，得到新的估计价值函数 $V$
   - d) 根据 $V$ 更新策略 $\pi$ 为贪心策略

## 第三讲 值函数估计

**26. 无模型强化学习的含义是什么？**

在现实问题中，状态转移和奖励函数通常未给出，只能观察到片段(episodes)。无模型强化学习直接从经验中学习值（value）和策略（policy），无需构建MDP模型。关键步骤：（1）估计值函数；（2）优化策略。

**27. 动态规划如何计算值函数？**

已知MDP时，通过Bellman等式迭代更新：$V^\pi(s) = R(s) + \gamma \sum_{s' \in S} P_{s\pi(s)}(s') V^\pi(s')$

**28. 蒙特卡洛方法的定义及举例？**

依赖于重复随机抽样获得数值结果。在强化学习中，用于估计值函数的均值替代期望：$V(s) \approx \frac{1}{N}\sum_{i=1}^N G_t^{(i)}$。例如：围棋胜率估计 = （以当前状态为起点模拟的胜利次数）/（总模拟次数）。

**29. 蒙特卡洛价值估计的实现流程？**

目标：从策略 $\pi$ 下的经验片段学习 $V^\pi$。
- 累计奖励：$G_t = R_{t+1} + \gamma R_{t+2} + \cdots + \gamma^{T-1} R_T$
- 增量更新：对片段中每个时间步状态 $S_t$：
  - $N(S_t) \leftarrow N(S_t) + 1$
  - $V(S_t) \leftarrow V(S_t) + \frac{1}{N(S_t)}(G_t - V(S_t))$
- 非稳态问题使用固定步长：$V(S_t) \leftarrow V(S_t) + \alpha(G_t - V(S_t))$

**30. 使用重要性采样的离线策略蒙特卡洛是什么？**

用策略 $\mu$ 产生的数据评估策略 $\pi$。对累积奖励按重要性比率加权：

$$G_t^{\pi/\mu} = \frac{\pi(a_t|s_t)}{\mu(a_t|s_t)} \cdot \frac{\pi(a_{t+1}|s_{t+1})}{\mu(a_{t+1}|s_{t+1})} \cdots G_t$$

更新：$V(s_t) \leftarrow V(s_t) + \alpha(G_t^{\pi/\mu} - V(s_t))$。缺点：显著增大方差。

**31. 使用重要性采样的离线策略时序差分是什么？**

只需一步重要性采样修正：

$$V(s_t) \leftarrow V(s_t) + \alpha \frac{\pi(a_t|s_t)}{\mu(a_t|s_t)}(r_{t+1} + \gamma V(s_{t+1}) - V(s_t))$$

比蒙特卡洛重要性采样方差更低，策略只需在单步中被近似。

**32. 时序差分学习（TD）的特点是什么？**

$$V(S_t) \leftarrow V(S_t) + \alpha(R_{t+1} + \gamma V(S_{t+1}) - V(S_t))$$

- 时序差分目标：$R_{t+1} + \gamma V(S_{t+1})$
- 时序差分误差：$\delta_t = R_{t+1} + \gamma V(S_{t+1}) - V(S_t)$
- 直接从经验片段学习，**模型无关**
- 通过**自举法（bootstrapping）**，从不完整片段中学习

**33. MC和TD的对比（优缺点）？**

| 特点 | 蒙特卡洛（MC） | 时序差分（TD） |
|---|---|---|
| 是否需要完整片段 | 是，必须等到终止 | 否，每步可更新 |
| 有偏/无偏 | 无偏估计 | 有偏估计（用估计值更新） |
| 方差 | 高（多步随机） | 低（单步随机） |
| 收敛性 | 收敛性好，函数近似也如此 | 有时不收敛（函数近似时） |
| 初始值敏感性 | 不敏感 | 更敏感 |
| 连续环境 | 只能用于片段化任务 | 可用于连续任务 |

**34. n步时序差分和n步累计奖励是什么？**

n步累计奖励：$G_t^{(n)} = R_{t+1} + \gamma R_{t+2} + \cdots + \gamma^{n-1} R_{t+n} + \gamma^n V(S_{t+n})$

n步TD学习：$V(S_t) \leftarrow V(S_t) + \alpha(G_t^{(n)} - V(S_t))$

当 $n=1$ 时为TD，当 $n=\infty$ 时为MC。

## 第四讲 无模型控制方法

**35. SARSA的含义和更新公式？**

SARSA是在线策略（on-policy）时序差分控制方法，对每个$(s, a, r, s', a')$元组更新：

$$Q(s,a) \leftarrow Q(s,a) + \alpha(r + \gamma Q(s', a') - Q(s,a))$$

- **策略评估**：SARSA更新Q值
- **策略改进**：ε-greedy策略改进
- 两个"A"都由当前策略选择

**36. Q学习的特点和更新表达式？**

Q学习是**离线策略（off-policy）**方法：

$$Q(s_t, a_t) \leftarrow Q(s_t, a_t) + \alpha\left(r_{t+1} + \gamma \max_{a'} Q(s_{t+1}, a') - Q(s_t, a_t)\right)$$

特点：无需重要性采样（直接学习Q(s,a)，相当于μ没有起作用）。收敛定理：Q学习控制收敛到最优状态-动作值函数 $Q(s,a) \to Q^*(s,a)$。

**37. $Q(s_t, a_t)$ 值更新表达式的含义？**

$r_{t+1}$：即时奖励；$\gamma \max_{a'} Q(s_{t+1}, a')$：下一状态的最优价值估计（折扣后）；两者之和为TD目标，减去当前估计后乘以步长 $\alpha$ 进行更新。

**38. 离线策略学习的含义及为什么使用？**

目标策略 $\pi(a|s)$ 用于价值函数评估，行为策略 $\mu(a|s)$ 用于收集数据。使用原因：平衡探索与利用、从人类/其他智能体观察学习、重用旧策略产生的经验。

**39. 了解Q学习收敛性证明的思路？**

定义收缩算子 $H$：$HQ = r(s,a) + \gamma \mathbb{E}_{s' \sim p(\cdot|s,a)}[\max_{a'} Q(s', a')]$。最优值函数 $Q^*$ 是 $H$ 的不动点（$Q^* = HQ^*$）。通过柯西数列论证任何初始 $Q$ 函数最终收敛到 $Q^*$。

**40. TD(λ)算法中的λ-累计奖励是什么？**

$$G_t^\lambda = (1-\lambda) \sum_{n=1}^\infty \lambda^{n-1} G_t^{(n)}$$

- $\lambda = 0$：等价于单步TD
- $\lambda = 1$：等价于蒙特卡洛
- 使用权重 $(1-\lambda)\lambda^{n-1}$ 对所有n步累计奖励加权平均

## 第五讲 规划与学习

**41. 策略值函数估计、策略提升含义？**

- **策略评估**：给定MDP和策略 $\pi$，计算 $V^\pi(s) = \mathbb{E}_{a \sim \pi(s)}[Q^\pi(s,a)]$
- **策略提升**：若对任何状态 $s$ 有 $Q^\pi(s, \pi'(s)) \geq V^\pi(s)$，则 $\pi'$ 是 $\pi$ 的策略提升

**策略提升定理**：若 $\pi'$ 满足上述条件，则对任何状态 $s$ 有 $V^{\pi'}(s) \geq V^\pi(s)$，即 $\pi'$ 比 $\pi$ 更优。

**42. 模型的含义、分类及作用？**

- **含义**：给定状态和动作，预测下一个状态和奖励的分布 $p(s', r|s, a)$
- **分类**：分布模型（描述所有可能性及概率）、样本模型（只产生一条轨迹）
- **作用**：产生模拟的经验数据 $\{s, a, r, s'\}$

**43. 规划的含义、通用框架、举例及优势？**

- **含义**：输入模型，输出策略的搜索过程
- **框架**：通过模型采样得到模拟数据 → 利用模拟数据更新值函数 → 改进策略
- **举例**：动态规划（搜索整个状态空间）
- **优势**：可在任何时间被打断或重定向；小而增量式的时间步规划在复杂问题中有效

**44. 规划和学习有什么不同？**

- **不同点**：规划用模型产生的模拟经验；学习用环境产生的真实经验
- **相同点**：都通过回溯（back-up）更新值函数估计；学习的方法可用在模拟经验上

**45. Dyna算法的经验不同用途、框架？**

经验的两种用途：（1）**更新模型**（间接强化学习，需求数据少）；（2）**更新值函数和策略**（直接强化学习，简单不受模型偏差影响）。

框架：
- 与环境交互 → 直接RL更新值函数和策略
- 与环境交互 → 学习模型
- 基于模型随机采样模拟经验 → 规划更新值函数和策略

**Dyna-Q算法流程**：在Q学习基础上，额外执行n次规划步骤（从模型中采样过去见过的 $S,A$，用模型预测 $R, S'$，执行Q学习更新）。

**Dyna-Q+的改进**：奖励改为 $r + \kappa\sqrt{\tau}$，其中 $\tau$ 是某状态多久未被到达，鼓励探索未访问的状态动作。

**46. 优先级采样的含义？**

根据值改变的幅度定义优先级 $P \leftarrow |R + \gamma \max_a Q(S', a) - Q(S, A)|$，使具有更大更新幅度的状态动作对被优先更新，收敛更快。

**47. 期望更新和采样更新对比？**

| | 期望更新 | 采样更新 |
|---|---|---|
| 需要 | 分布模型 | 采样模型 |
| 计算量 | 大 | 小 |
| 偏差 | 无偏（更准确） | 有采样误差 |
| 适用 | 小分支因子 | 大分支因子、多状态 |

**48. 轨迹采样的优缺点？**

- **优点**：不需要知道当前策略下状态的分布，计算量少，简单有效
- **缺点**：不断重复更新已经被访问的状态，忽略许多无关状态

**49. 实时动态规划（RTDP）和决策时规划的区别？**

- **RTDP**：实时轨迹采样，只更新轨迹访问的状态值，能跳过策略无关状态，满足条件时可以概率1收敛
- **决策时规划**：只着眼于当前状态的动作选择，为当前状态选一个动作（如启发式搜索、Rollout算法、蒙特卡洛树搜索）

**50. 了解启发式搜索、Rollout算法、蒙特卡洛树搜索？**

- **启发式搜索**：从当前状态展开树结构，叶节点代表估计值函数，回溯到当前状态，看多步规划下的最优行动
- **Rollout算法**：从当前状态进行蒙特卡洛估计，选取最高估计值动作，在下一状态重复（时间复杂度 $O(K \cdot A \cdot T_{eval})$）
- **MCTS**：选择→扩展→模拟（rollout）→回溯，保留部分经验，应用于AlphaGo

## 第六讲 价值和策略近似逼近方法

**51. 离散化连续马尔可夫决策过程，分桶的含义及优缺点？**

对连续状态空间使用网格切分或分桶（将相似状态归类），得到离散 $\hat{S}$，转化为 $(\hat{S}, A, P_{\hat{s}a}, \gamma, R)$。

- **优点**：操作简洁直观、高效
- **缺点**：过于简单，可能假设每个离散区间常数值；**维度灾难** $|S| = R^n \Rightarrow |\hat{S}| = \{1,\ldots,k\}^n$

**52. 值函数近似的主要形式？**

$V_\theta(s) \approx V^\pi(s)$，$Q_\theta(s,a) \approx Q^\pi(s,a)$，其中 $\theta$ 可通过强化学习更新。可用线性模型、神经网络等可微函数（因为需要做梯度优化）。

**53. 随机梯度下降（SGD）的值函数近似？**

目标：最小化 $J(\theta) = \mathbb{E}_\pi[\frac{1}{2}(V^\pi(s) - V_\theta(s))^2]$

$$\theta \leftarrow \theta + \alpha(V^\pi(s) - V_\theta(s))\frac{\partial V_\theta(s)}{\partial \theta}$$

**54. 蒙特卡洛状态值函数近似？**

用累积奖励 $G_t$ 作为真实目标值，在"训练数据" $(s_1, G_1), (s_2, G_2), \ldots$ 上做监督学习：

$$\theta \leftarrow \theta + \alpha(G_t - V_\theta(s))\frac{\partial V_\theta(s_t)}{\partial \theta}$$

**55. 时序差分的状态-动作值函数近似？**

$$\theta \leftarrow \theta + \alpha(r_{t+1} + \gamma Q_\theta(s_{t+1}, a_{t+1}) - Q_\theta(s,a))\frac{\partial Q_\theta(s,a)}{\partial \theta}$$

注意：虽然 $\theta$ 出现在目标中，但不计算目标函数的梯度（半梯度方法）。

**56. 参数化策略的含义？**

将策略参数化：$\pi_\theta(a|s) = P(a|s; \theta)$，策略可以是确定性的 $a = \pi_\theta(s)$ 或随机的。将可见状态泛化到未知状态上。

**57. 基于策略的强化学习的优缺点？**

- **优点**：更好的收敛性；在高维度或连续动作空间更有效；能学习随机策略
- **缺点**：通常收敛到局部最优；策略评估不够高效且方差大

**58. 策略梯度定理？**

对任意可微策略 $\pi_\theta(a|s)$ 和策略目标函数 $J$（包括起始状态、平均奖励、平均价值等目标）：

$$\frac{\partial J(\theta)}{\partial \theta} = \mathbb{E}_{\pi_\theta}\left[\frac{\partial \log \pi_\theta(a|s)}{\partial \theta} Q^{\pi_\theta}(s,a)\right]$$

**59. 蒙特卡洛策略梯度算法（REINFORCE）？**

利用累积奖励 $G_t$ 作为 $Q^{\pi_\theta}(s,a)$ 的无偏采样，更新：

$$\Delta\theta_t = \alpha \frac{\partial \log \pi_\theta(a_t|s_t)}{\partial \theta} G_t$$

**60. Softmax随机策略？**

$$\pi_\theta(a|s) = \frac{e^{f_\theta(s,a)}}{\sum_{a'} e^{f_\theta(s,a')}}$$

其log形式梯度：$\frac{\partial \log \pi_\theta(a|s)}{\partial \theta} = \frac{\partial f_\theta(s,a)}{\partial \theta} - \mathbb{E}_{a' \sim \pi_\theta(a'|s)}\left[\frac{\partial f_\theta(s,a')}{\partial \theta}\right]$

**61. REINFORCE存在的问题、Actor-Critic的思想？**

REINFORCE的问题：基于片段式数据（需要终止状态）、低数据利用效率、高训练方差（最重要缺陷）。

Actor-Critic思想：为什么不建立一个可训练的值函数 $Q_\Phi$ 来估计 $G_t$？
- **演员（Actor）** $\pi_\theta(a|s)$：采取动作使评论家满意的策略
- **评论家（Critic）** $Q_\Phi(s,a)$：学习准确估计演员策略动作价值的值函数

**62. A2C算法？**

优势函数：$A^\pi(s,a) = Q^\pi(s,a) - V^\pi(s)$

由于 $Q^\pi(s,a) \approx r(s,a) + \gamma V^\pi(s')$，所以 $A^\pi(s,a) \approx r(s,a) + \gamma V^\pi(s') - V^\pi(s)$，只需拟合 $V$ 函数即可。减去基线降低方差，同时提供更多信息（降低较差动作概率，提高较优动作概率）。

## 第七讲 深度强化学习价值方法

**63. 深度强化学习的思想及特点？**

利用深度神经网络进行价值函数和策略近似，使RL算法能以端到端方式解决复杂问题。比单独的RL和DL都更难训练（高维参数空间、训练不稳定、数据需求大）。

**64. 深度强化学习的分类？**

- **基于价值的方法**：深度Q网络（DQN）及其扩展
- **基于随机策略的方法**：策略梯度、TRPO、PPO、A3C
- **基于确定性策略的方法**：DPG、DDPG

**65. 深度Q网络（DQN）的思想及解决办法、算法流程？**

**思想**：使用神经网络逼近 $Q_\theta(s,a)$，但直接使用不稳定。

**解决办法**：
1. **经验回放**：存储 $e_t = (s_t, a_t, s_{t+1}, r_t)$ 于数据库 $D$ 中，采样时服从均匀分布（打破数据相关性）
2. **目标网络 $Q_{\theta^-}$**：使用较旧参数，每隔 $C$ 步与训练网络同步（稳定目标值）

**算法流程**：
1. 收集数据：用ε-greedy探索，将 $(s_t, a_t, s_{t+1}, r_t)$ 放入经验池
2. 采样：从数据库中采样 $k$ 个样本
3. 更新网络：计算Loss，更新 $Q$ 网络参数 $\theta$；每 $C$ 次迭代更新目标网络 $\theta^-$

损失函数：$L_i(\theta_i) = \mathbb{E}_{s,a,r,s' \sim U(D)}\left[\left(r + \gamma \max_{a'} Q(s', a'; \theta_i^-) - Q(s, a; \theta_i)\right)^2\right]$

**66. Q-learning中的过估计、Double DQN？**

**过估计原因**：$\mathbb{E}[\max(X_1, X_2)] \geq \max(\mathbb{E}[X_1], \mathbb{E}[X_2])$，max操作使Q值越来越大甚至超过真实值，且动作数越多越严重。

**Double DQN**：解耦合动作选择和价值估计，用不同网络分别负责：

$$y_t = r_t + \gamma Q_{\theta'}(s_{t+1}, \arg\max_{a'} Q_\theta(s_{t+1}, a'))$$

（行为网络 $\theta$ 选择动作，目标网络 $\theta'$ 评估价值）

**67. Dueling DQN的改进措施、网络结构、优点？**

**核心思想**：$Q(s,a) = V(s) + A(s,a)$，其中 $A^\pi(s,a) = Q^\pi(s,a) - V^\pi(s)$ 是优势函数。

**网络结构**：共享卷积层 → 分为两支（$V(s;\theta,\beta)$ 和 $A(s,a;\theta,\alpha)$） → 聚合：

$$Q(s,a;\theta,\alpha,\beta) = V(s;\theta,\beta) + \left(A(s,a;\theta,\alpha) - \frac{1}{|A|}\sum_{a'} A(s,a';\theta,\alpha)\right)$$

**优点**：可在不考虑动作的情况下判断状态好坏；状态值函数学习更有效（一个 $V$ 对应多个 $A$）。

## 第八讲 深度强化学习策略方法

**68. 策略网络的梯度，策略梯度和Q学习的对比？**

| | Q学习 | 策略梯度 |
|---|---|---|
| 学习对象 | $Q_\theta(s,a)$ | $\pi_\theta(a|s)$ |
| 优化目标 | 最小化TD误差 | 直接最大化策略价值 |
| 更新方向 | 梯度下降 | 梯度上升 |
| 目标函数 | $\min J(\theta) = \mathbb{E}[(r + \gamma\max Q_{\theta'} - Q_\theta)^2]$ | $\max J(\theta) = \mathbb{E}_{\pi_\theta}[A^{\pi_\theta}(s,a)]$ |

**69. A3C和A2C的改进、A3C框架图？**

A3C = 异步（Asynchronous）+ 优势（Advantage）+ 动作评价（Actor-Critic）。

**改进**：异步并行执行多个环境，多个worker独立与各自环境交互，梯度异步更新全局网络，有效利用多核CPU资源。

优势函数：$A(s_t, a_t; \theta_v) = \sum_{i=0}^{k-1} \gamma^i r_{t+i} + \gamma^k V(s_{t+k}; \theta_v) - V(s_t; \theta_v)$

**70. 策略梯度的缺点？**

步长难以确定：采集到的数据分布随策略更新而变化，步长太大时会导致更新后策略变差，进而新采集数据变坏，形成恶性循环。

**71. TRPO原理、θ更新的表达式和过程？**

使用KL散度约束策略更新的幅度：

$$\theta' \leftarrow \arg\max_{\theta'} \sum_t \mathbb{E}_{s_t \sim p_\theta, a_t \sim \pi_\theta}\left[\frac{\pi_{\theta'}(a_t|s_t)}{\pi_\theta(a_t|s_t)} \gamma^t A^{\pi_\theta}(s_t, a_t)\right]$$

$$\text{such that } \mathbb{E}_{s_t \sim p}[D_{KL}(\pi_{\theta'}(a_t|s_t) \| \pi_\theta(a_t|s_t))] \leq \epsilon$$

步骤：（1）优化上式更新 $\theta'$；（2）更新 $\lambda \leftarrow \lambda + \alpha(D_{KL} - \epsilon)$。需要共轭梯度法求解。

**72. TRPO的不足，PPO和TRPO的不同与改进措施？**

**TRPO不足**：近似带来误差、求解约束优化问题困难。

**PPO改进**：
1. **截断式优化目标**：$L^{CLIP}(\theta) = \mathbb{E}_t[\min(r_t(\theta)\hat{A}_t, \text{clip}(r_t(\theta), 1-\epsilon, 1+\epsilon)\hat{A}_t)]$，构建优化目标下界，避免步子过大
2. **优势函数用多步TD**：$\hat{A}_t = -V(s_t) + r_t + \gamma r_{t+1} + \cdots + \gamma^{T-t} V(s_T)$
3. **自适应KL惩罚项**：动态调整 $\beta$

PPO更简洁、更简单、实验效果更好，是目前最常用的深度策略梯度算法。

**73. DPG（确定性策略梯度）？**

对于连续动作，确定性策略 $a = \pi_\theta(s)$ 可微，梯度为：

$$\nabla_\theta J(\pi_\theta) = \mathbb{E}_{s \sim \rho^\pi}[\nabla_\theta \pi_\theta(s) \nabla_a Q^w(s,a)|_{a=\pi_\theta(s)}]$$

**74. DDPG（深度确定性策略梯度）相对于DPG的改进及算法流程？**

**改进**：在DPG基础上加入：（1）经验重放（离线策略）；（2）目标网络；（3）在动作输入前批标准化Q网络；（4）添加连续噪声用于探索。

**算法流程**：收集数据（带噪声动作） → 存入经验池 → 采样 → 更新Critic（Q网络） → 更新Actor（策略网络） → 延迟更新目标网络。

## 第九讲 基于模型的深度强化学习

**75. 基于模型的强化学习的特点？**

使用环境模型（模拟器）替代真实环境，从模型中采样模拟数据 $\{(s,a,r,s')\}$ 训练智能体策略，提升样本效率。模型包括：状态转移动态 $\hat{p}(s'|s,a)$ 和奖励函数 $\hat{r}(s,a)$。

**76. 模型预测控制（MPC）？**

在每个时间步，算法采样多条轨迹，从这些轨迹中得到经验最优轨迹，取该轨迹的第一个动作来执行，如此往复。不需要学习值函数和策略，直接基于模型工作。

**77. 打靶法的算法流程、优缺点、改进方法？**

**流程**：从 $s_0$ 出发，对每个候选 $a_0$ 附上随机动作序列 $[a_0,a_1,\ldots,a_T]$，用模型交互得到轨迹，计算经验价值 $\hat{Q}(s_0,a_0) = \sum_t \gamma^t \hat{r}_t$，选最高价值动作执行。

**优点**：实现简单，无训练开销，不需要预先设置时间步。

**缺点**：不确定性高（方差大），可能无法选到最优动作。

**改进**：交叉熵方法（CEM）——维护带来较高奖励的动作分布，从该分布中采样。

**78. PETS算法中PE、TS的含义及两种不确定性？**

PETS = Probabilistic Ensembles with Trajectory Sampling。

- **PE（概率集成）**：用多个高斯过程神经网络构成集成，刻画两种不确定性：
  - **认知不确定性（epistemic uncertainty）**：不同单模型之间的不一致性，反映数据缺少区域的模型不确定性
  - **偶然不确定性（aleatoric uncertainty）**：每个高斯过程建模真实随机环境的固有随机性
- **TS（轨迹采样）**：每次预测从 $N$ 个模型中挑选一个，一条轨迹采样使用多个环境模型

**79. 分支推演的含义、对策略价值误差的分析、MBPO算法要点？**

**分支推演（Branched Rollout）**：从真实轨迹中随机选择状态作为起点，用环境模型和当前策略做交互，采样 $k$ 步得到分支轨迹，与真实数据共同训练策略。

**策略价值误差分析**：$|\eta(\pi) - \hat{\eta}(\pi)|$ 与模型泛化误差和策略偏移程度有关，随 $k$ 增大：误差因策略偏移增大，但样本效率提升。当模型泛化误差/策略偏移程度比率足够小时，$k > 0$ 有利。

**MBPO算法要点**：
- 分支起点从真实经验轨迹中随机采样任何状态（非必须从初始状态）
- 分支步数 $k$ 取决于环境模型和策略质量
- 得到推演数据后，使用Soft AC算法训练策略

## 第十讲 模仿学习

**80. 模仿学习的特点、示例、难点？**

**特点**：从专家示例中学习策略，仅给出专家轨迹，奖励函数未知。

**示例**：自动驾驶、机器人控制。

**难点**：某些任务难以定义奖励函数；人为设置的奖励函数可能导致不合理行为。

**81. 模仿学习与监督学习的比较？**

模仿学习需要考虑：（1）解的结构性特征（约束、动态平滑性、多步计划连贯性）；（2）决策者决策和输入分布之间的关系（在线/离线策略）；（3）数据收集成本高，需最小化。

**82. 三类模仿学习算法？**

1. **行为克隆（Behavior Cloning）**：无需奖励函数，学习状态→动作的直接映射
2. **逆强化学习（Inverse RL）**：找到使专家策略比其他策略更优的奖励函数
3. **生成对抗模仿学习（GAIL）**：通过GAN方式自动学习奖励函数

**83. 行为克隆的分布偏移、DAgger技术？**

**分布偏移**：策略执行时遇到的状态分布与训练时看到的分布不匹配，错误会积累放大。

**DAgger（Dataset Aggregation）**：从稳定的控制器学习策略（有噪声），加入更多在线策略数据，迭代地将新状态下的专家标注加入训练集。

**其他挑战**：不具有马尔可夫性质的行为（依赖历史观测）、多模态行为。

**84. 逆强化学习：正逆强化学习对比、最大因果熵IRL？**

| | 正向强化学习 | 逆强化学习 |
|---|---|---|
| 给定 | 状态、动作、奖励函数 | 状态、动作、专家轨迹 |
| 学习 | 最优策略 $\pi^*(a|s)$ | 奖励函数 $r_\psi(s,a)$，再学习 $\pi^*$ |

**最大因果熵IRL步骤**：
1. 找到使专家策略产生较低代价、其他策略产生较高代价的代价函数 $\hat{c}$
2. 通过使用代价函数 $\hat{c}$ 进行正向RL，恢复专家策略

**优势**：使用神经网络计算成本函数。**局限性**：仍需反复求解MDP，假设环境已知。

**85. 生成对抗模仿学习（GAIL）：与GAN的关系，判别器D的作用？**

$$\min_\pi \max_D \mathbb{E}_{(s,a) \sim \pi_E}[\log D(s,a)] + \mathbb{E}_{(s,a) \sim \pi}[\log(1-D(s,a))] - \lambda H(\pi)$$

判别器 $D$ 的作用：判别数据是由当前策略 $\pi$（生成器）还是专家 $\pi_E$（真实数据）产生的，为策略提供奖励信号。

**86. 生成对抗网络与逆强化学习之间的联系？**

两者结构相似：GAN = 生成器（欺骗判别器）+ 判别器（区分真假）；IRL = 代价函数学习 + 基于代价函数的策略优化。GAIL将两者统一，判别器充当奖励函数，策略充当生成器。

## 第十一讲 离线强化学习

**87. 离线强化学习的动机、定义、优势？**

**动机**：真实环境从零训练风险高（无人驾驶、医疗）且昂贵（机器人控制）。

**定义**：在给定的离线数据集上直接训练智能体策略，训练过程中不得与环境交互。

**优势**：预训练强化学习策略；评测策略好坏；缩小学术研究与真实落地的差距；使RL更像有监督学习。

**88. 在线策略学习、离线强化学习、离线策略学习的不同？**

| | 在线策略 | 离线策略 | 离线强化学习 |
|---|---|---|---|
| 训练时与环境交互 | 是 | 是 | **否** |
| 数据是否来自别的策略 | 否 | 是 | 是 |

**89. 外延误差（Extrapolation Error）问题？**

**定义**：智能体如果涉足到了从没有见过的、远离数据集的状态动作对（OOD问题），Q值估计会出现严重误差，并随TD公式传播到非OOD数据上。

**根本原因**：数据集分布（收集策略 $\mu$ 的占用度量）和当前策略（$\pi$ 的占用度量）不一致。

**90. 离线强化学习的分类？**

- **无模型方法**：显式限制（BCQ、BEAR、BRAC、CQL）、隐式限制（AWR、REM）
- **基于模型的方法**：使用学习的模型估计不确定性（MORel、MOPO、COMBO）

**91. BCQ算法的基本思路、实现方法、算法大致流程？**

**基本思路**：仅使用在数据集支撑上的目标Q值做时序差分计算，避免OOD动作。

**实现方法**：使用变分自动编码器（VAE）生成距离数据集较近的状态动作对，加上 $[-\Phi, +\Phi]$ 的小扰动。

**关键权衡**：$n$（采样数）和 $\Phi$（扰动幅度）越小越接近模仿学习，越大越接近强化学习但易出OOD问题。

**92. CQL算法的思路、目标函数的三部分构成、正则项？**

**思路**：学习一个保守的、可作为价值下界的Q函数，避免在OOD数据上的过高估计。

**目标函数三部分**：
1. 新策略 $\mu$ 的数据Q价值惩罚（降低OOD Q值）
2. 数据集支撑范围内的Q价值奖励（提升数据内Q值）
3. 时序差分误差（标准RL目标）

**正则项**：使用 $\mu$ 和均匀分布之间的KL散度作为正则项（经验性实现）。

## 第十二讲 参数化行动空间

**93. 离散和连续行动空间对比？**

| | 离散动作空间 | 连续动作空间 |
|---|---|---|
| 表示 | $a \in \{a_1, a_2, \ldots, a_k\}$ | $a = (x_1, x_2, \ldots, x_d)$ |
| 问题类型 | 分类问题 | 回归问题 |
| 局限 | 无法对选择的动作微调 | 无法考虑动作种类差异 |
| 适用算法 | DQN, A3C | DPG, DDPG |

**94. 参数化动作空间的含义、举例？**

每个离散动作都被赋予连续参数：$\{$冲刺(方向,速度), 转向(方向), 踢球(方向,力度), 停止$\}$。混合动作空间，智能体同时决定执行何种离散动作及连续参数。

**95. 参数化动作马尔可夫决策过程（PAMDPs）？**

混合动作空间表示：$A = \{(k, x_k) | x_k \in X_k \text{ for all } k \in \{1\ldots K\}\}$。强化学习方法分类：离散化连续动作、放宽离散动作空间、分开处理离散和连续动作。

**96. 处理离散-连续混合动作空间的算法途径？**

1. **离散化连续动作空间**：表示为 $A = \{(k,x_k)|x_k \in X_k\}$，用DQN/A3C训练，缺点是丢失自然结构，维度爆炸
2. **放宽离散动作空间**：输出 $(f_1,\ldots,f_K, x_1,\ldots,x_K)$，用DDPG训练连续策略
3. **分开处理**：如Q-PAMDP、PDQN

**97. 离散化连续动作空间的表示方法、适用算法、缺点？**

用离散子集近似每个 $X_k$，使用DQN或A3C。缺点：丢失自然结构；良好近似需要大量离散动作；维度灾难。

**98. 放宽离散动作空间，如何选择离散动作？适用算法（DDPG模型）？**

策略输出 $(f_1,\ldots,f_K, x_1,\ldots,x_K)$，离散动作选择：确定性地取 $\arg\max_i(f_i)$ 或随机地按 $\text{softmax}(f_i)$。Q值网络（Critic）平滑处理离散的argmax操作。

**99. Q-PAMDP算法特点、大致流程？**

交替学习：固定 $\theta$（参数策略），用Q学习优化离散策略 $\pi_w^d$；固定 $w$（离散策略），用策略搜索优化参数策略 $\pi_\theta^a$；循环直至收敛。

**100. 参数化深度Q网络（PDQN）的思路、与DDPG的对比？**

思路：DQN（选离散动作）+ DDPG（确定连续参数）的结合。

损失函数：Critic损失 $\ell_t^Q(w) = \frac{1}{2}(Q(s_t, k_t, x_{k_t}; w) - y_t)^2$；Actor损失 $\ell_t^\Theta(\theta) = -\sum_{k=1}^K Q(s_t, k, x_k(s_t;\theta);w_t)$

## 第十三讲 目标导向的强化学习

**101. 适用的强化学习范围特点？**

大部分RL方法只能完成特定任务（固定地图、起点、终点）。适用于需要完成不同目标（起点/终点组合）的复杂任务，以及可分解成多个子任务的层次化任务。

**102. 目标导向强化学习（GoRL）的思路、含义？**

将每次任务的目标显式加入智能体的输入和奖励信号。GoRL在MDP环境中额外定义一个目标 $g$，显式告诉智能体，在训练过程中使用目标信息，使得测试阶段给定新目标时智能体能直接泛化。

**103. 目标导向强化学习的问题建模（符号约定、与一般RL的不同、智能体目标）？**

扩展元组：$(S, A, p, \gamma, r_g, G, \phi)$，其中 $G$ 是目标集合，$\phi: S \to G$ 将状态映射为目标，$r_g: S \times A \times G \to \mathbb{R}$ 是目标导向奖励函数，$\pi_i: S \times G \to \Omega(A_i)$ 是目标导向策略。

**与一般RL的不同**：策略和奖励函数都加入了目标信息。

**智能体目标**：最大化期望累积奖励 $J(\pi) = \mathbb{E}[\sum_t \gamma^t r_g(s_t, a_t, s_{t+1})]$

**104. 具体模块定义（目标导向奖励函数、策略、普适价值函数逼近）？**

- **目标导向奖励函数**：通常为 $r_g(s,a) = \mathbb{1}[\phi(s) = g]$（一个较小阈值内达到目标为1，否则为0）
- **目标导向策略**：$\pi(a|s,g)$，简单实现可以拼接 $s$ 和 $g$ 作为输入
- **普适价值函数逼近（UVFA）**：$V: S \times G \to \mathbb{R}$，最简单在MLP输入层拼接 $s$ 和 $g$

**105. 经典GoRL算法HER的做法和大致算法流程？**

**问题**：GoRL奖励极其稀疏，训练初期难以得到奖励。

**HER（Hindsight Experience Replay）思路**：使用重标注（relabeling）有效利用"失败"经验——将失败轨迹中某个状态 $s''$ 作为新目标 $g' = \phi(s'')$，这条轨迹就变成了以 $g'$ 为目标的"成功"经验。

**算法流程**：
1. 初始化策略参数 $\theta$ 和经验回放池 $R$
2. 对每个序列：用策略采样轨迹，以 $(s,a,r,s',g)$ 形式存入 $R$
3. 从 $R$ 采样；选择状态 $s''$，令 $g' = \phi(s'')$ 计算新奖励 $r'$，用 $(s,a,r',s',g')$ 替换原元组
4. 用新元组训练策略

注：HER框架适合各种RL方法（DQN、DDPG、PPO）。

## 第十四讲 多智能体强化学习

**106. 多智能体强化学习的特点？**

环境包含不断学习更新的其他智能体，在任何单个智能体视角下环境是**非稳态的（non-stationary）**（环境迁移分布会发生改变）。智能体不仅与环境交互，还要与其他智能体交互。

**107. 多智能体学习的难点？**

将其他智能体视为环境的一部分使用单智能体Q学习，会破坏理论上的收敛保证（一个智能体策略的改变会影响其他智能体策略，环境非稳态）。理论上更加困难：单智能体（模型→学习目标→数据）vs. 多智能体（模型↔学习目标↔数据，相互影响）。

**108. 博弈论视角下序贯决策的三种问题？**

1. **马尔可夫决策过程**：一个智能体 + 多个状态
2. **重复博弈**：多个智能体 + 一个状态
3. **随机博弈（马尔可夫博弈）**：多个智能体 + 多个状态（MARL的主要框架）

**109. 随机博弈的问题建模（博弈表格、状态转移、奖励函数、价值函数）？**

元组 $(N, S, A, R, p, \gamma)$：
- $N$ 个智能体，联合策略 $\pi = (\pi_1, \ldots, \pi_N)$，联合动作 $a = (a_1, \ldots, a_N)$
- 状态转移 $p: S \times A \to \Omega(S)$
- 各智能体奖励函数 $r_i$ 在博弈表格中，每个状态对应一个博弈表格，智能体同时选择动作
- 价值函数：$V_i^\pi(s) = \sum_{t=0}^\infty \gamma^t \mathbb{E}_{\pi,p}[r_i^t|s_0=s]$
- Q值函数：$Q_i^\pi(s, a) = r_i(s,a) + \gamma \mathbb{E}_{s' \sim p}[V_i^\pi(s')]$

**110. 多智能体强化学习的方法分类？**

1. **完全中心化**：所有智能体状态聚合为全局状态，所有动作连接为联合动作，优点：环境稳态；缺点：复杂度高
2. **完全去中心化**：每个智能体独立学习，假设其他智能体策略不变；优点：简单；缺点：不收敛
3. **中心化训练去中心化执行（CTDE）**：训练时用全局信息，执行时用局部信息

**111. 完全中心化方法：整体智能体的特点和缺点、纳什Q学习的操作步骤及缺点？**

**整体智能体**：适合合作任务，奖励为各智能体之和。缺点：动作空间太大，状态转移和奖励函数复杂度高；无法处理非合作任务。

**纳什Q学习**：基于当前Q值表求解纳什均衡 $\pi^*$ 和 $V_{Nash}$，再用纳什均衡价值更新Q值表，如此循环直至收敛。缺点：计算复杂度极高，无法处理非合作博弈。

**112. 完全去中心化方法：优缺点、独立Q学习与独立PPO？**

- **优点**：简单好实现
- **缺点**：环境非稳态，很可能不收敛

**独立Q学习**：对智能体 $i$，假设其他策略不变，直接做Q学习（但奖励信号实际基于联合动作）。

**独立PPO**：对每个智能体做PPO学习（可共享参数）。

**113. CTDE算法MADDPG的特点及流程？**

**特点**：
- 每个智能体是actor-critic
- **Actor** $\mu_i(a_i|o_i)$：自己独有，只看局部观测
- **Critic** $Q_i^\mu(x, a_1, \ldots, a_N)$：所有智能体共享，有全局信息 $x = (o_1, \ldots, o_N)$

**流程**：
1. 随机初始化各智能体Actor和Critic网络
2. 获取初始观测 $x$
3. 每个智能体用策略选动作（加探索噪声）
4. 执行联合动作，获得奖励和新观测
5. 存入经验回放池 $D$
6. 采样数据：中心化训练Critic（TD更新）、训练自身Actor（策略梯度）、更新目标网络

## 附录：核心公式汇总

| 算法 | 核心更新公式 |
|---|---|
| 蒙特卡洛 | $V(S_t) \leftarrow V(S_t) + \alpha(G_t - V(S_t))$ |
| TD(0) | $V(S_t) \leftarrow V(S_t) + \alpha(R_{t+1} + \gamma V(S_{t+1}) - V(S_t))$ |
| SARSA | $Q(s,a) \leftarrow Q(s,a) + \alpha(r + \gamma Q(s',a') - Q(s,a))$ |
| Q-Learning | $Q(s_t,a_t) \leftarrow Q(s_t,a_t) + \alpha(r_{t+1} + \gamma\max_{a'} Q(s_{t+1},a') - Q(s_t,a_t))$ |
| 策略梯度 | $\nabla_\theta J(\theta) = \mathbb{E}_{\pi_\theta}[\nabla_\theta \log\pi_\theta(a|s) Q^{\pi_\theta}(s,a)]$ |
| A2C优势 | $A^\pi(s,a) = r(s,a) + \gamma V^\pi(s') - V^\pi(s)$ |
| Double DQN目标 | $y_t = r_t + \gamma Q_{\theta'}(s_{t+1}, \arg\max_{a'} Q_\theta(s_{t+1},a'))$ |
| PPO截断 | $L^{CLIP} = \mathbb{E}[\min(r_t\hat{A}_t, \text{clip}(r_t, 1-\epsilon, 1+\epsilon)\hat{A}_t)]$ |
| DDPG策略梯度 | $\nabla_\theta J = \mathbb{E}[\nabla_\theta \pi_\theta(s)\nabla_a Q^\pi(s,a)|_{a=\pi_\theta(s)}]$ |
