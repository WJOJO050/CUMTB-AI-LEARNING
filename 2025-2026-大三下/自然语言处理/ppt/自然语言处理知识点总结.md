## 第1章 绪论

### 1.1 基本概念

| 概念 | 定义 |
|------|------|
| **语言** | 人类用来表达意思、交流思想的工具，由语音、词汇和语法构成的系统 |
| **自然语言** | 人类社会发展过程中自然产生的语言（区别于程序语言等人造语言） |
| **语言学** | 对语言的科学研究 |
| **语音学** | 研究人类发音特点，提出语音描述、分类和转写方法的科学 |
| **自然语言理解 (NLU)** | 从自然语言到机器内部的映射；机器能够执行人类期望的语言功能 |
| **自然语言处理 (NLP)** | 研究人与人交际中以及人与计算机交际中的语言问题的学科 |
| **计算语言学** | 利用计算机进行语言分析的学科 |

### 1.2 语言类型

| 语言类型 | 特点 | 代表语言 |
|----------|------|----------|
| **曲折语** | 用词的形态变化表示语法关系 | 英语、法语、德语 |
| **黏着语** | 词内有专门表示语法意义的附加成分 | 日语 |
| **孤立语(分析语)** | 形态变化少，靠词序和虚词表示语法关系 | 汉语 |

### 1.3 NLP发展历史

```
基础研究(1950) → 规则系统(1980) → 统计方法(1990) → 深度学习(2010)
```

**关键里程碑：**
- **1946年**：第一台电子计算机ENIAC
- **1954年**：Georgetown大学实现第一个MT系统（俄译英）

| 研究方向 | 说明 |
|----------|------|
| **机器翻译 (MT)** | 实现一种语言到另一种语言的自动翻译 |

### 1.5 基本研究方法

## 第2章 数学基础

### 2.1 概率论基础

- **概率**: $0 \leq P(A) \leq 1$，$P(\Omega) = 1$
- **条件概率**: $P(A|B) = \frac{P(A \cap B)}{P(B)}$
- **贝叶斯公式**: $P(A|B) = \frac{P(B|A)P(A)}{P(B)}$

### 2.2 信息论基础

#### 熵 (Entropy)
$$H(X) = -\sum_{x \in X} p(x) \log_2 p(x)$$

- 熵表示随机变量的不确定性
- 熵越大，不确定性越大

#### 联合熵 (Joint Entropy)
$$H(X, Y) = -\sum_{x \in X}\sum_{y \in Y} p(x, y) \log_2 p(x, y)$$

#### 条件熵 (Conditional Entropy)
$$H(X|Y) = -\sum_{x \in X}\sum_{y \in Y} p(x, y) \log_2 p(x|y)$$

#### 相对熵 (KL散度)
$$D(p||q) = \sum_{x \in X} p(x) \log \frac{p(x)}{q(x)}$$

- 用于衡量两个概率分布的差距
- 当两个分布相同时，相对熵为0

#### 交叉熵 (Cross Entropy)
$$H(X, q) = -\sum_{x} p(x) \log q(x)$$

- 衡量估计模型与真实概率分布之间的差异

#### 困惑度 (Perplexity)
$$PP_q = 2^{H(L,q)} = [q(x_1^n)]^{-\frac{1}{n}}$$

- 语言模型设计的目标是使困惑度最小
- 对于英语文本，困惑度范围一般为 50～1000

#### 互信息 (Mutual Information)
$$I(X; Y) = H(X) - H(X|Y) = \sum_{x \in X}\sum_{y \in Y} p(x, y) \log \frac{p(x, y)}{p(x)p(y)}$$

- 表示知道Y后X不确定性的减少量

## 第3章 形式语言与自动机

### 3.1 形式文法定义

文法 $G = (N, \Sigma, P, S)$

| 符号 | 含义 |
|------|------|
| $N$ | 非终结符集合 |
| $\Sigma$ | 终结符集合（字母表） |
| $P$ | 产生式规则集合 |
| $S$ | 起始符号 |

### 3.2 文法类型（乔姆斯基层次）

| 类型 | 名称 | 产生式形式 | 识别器 |
|------|------|------------|--------|
| **0型** | 无约束文法 | $\alpha \to \beta$ | 图灵机 |
| **1型** | 上下文有关文法 (CSG) | $\alpha A \beta \to \alpha \gamma \beta$ | 线性带限自动机 |
| **2型** | 上下文无关文法 (CFG) | $A \to \alpha$ | 下推自动机 |
| **3型** | 正则文法 | $A \to Bx$ 或 $A \to x$ | 有限自动机 |

**包含关系**: $L(G_0) \supseteq L(G_1) \supseteq L(G_2) \supseteq L(G_3)$

### 3.3 上下文无关文法 (CFG)

**推导方式：**
- **最左推导**: 每步只改写最左边的非终结符
- **最右推导（规范推导）**: 每步只改写最右边的非终结符

**派生树（语法树）：**
- 根节点为起始符号 $S$
- 内部节点为非终结符
- 叶子节点为终结符

**二义性：** 如果某个句子有不止一棵分析树，则该文法是二义的。

### 3.4 自动机

| 文法类型 | 自动机 | 特点 |
|----------|--------|------|
| 正则文法 | 有限自动机 (FA) | 最简单，识别正则语言 |
| 上下文无关文法 | 下推自动机 (PDA) | 增加栈存储 |
| 上下文有关文法 | 线性带限自动机 | 带子长度与输入成比例 |
| 0型文法 | 图灵机 (TM) | 最强大，可计算任何可计算函数 |

## 第5章 概率语法与语言模型

### 5.1 n-gram 语言模型

**核心思想**: 一个词的出现只依赖于它前面 $n-1$ 个词

$$p(w_i | w_1^{i-1}) \approx p(w_i | w_{i-n+1}^{i-1})$$

| 模型 | 名称 | 依赖词数 |
|------|------|----------|
| n=1 | Unigram | 0（独立） |
| n=2 | Bigram | 1（1阶马尔可夫链） |
| n=3 | Trigram | 2（2阶马尔可夫链） |

**句子概率计算：**
$$p(s) = \prod_{i=1}^{m+1} p(w_i | w_{i-n+1}^{i-1})$$

### 5.2 参数估计

**最大似然估计 (MLE):**
$$p(w_i | w_{i-n+1}^{i-1}) = \frac{c(w_{i-n+1}^{i})}{c(w_{i-n+1}^{i-1})}$$

其中 $c(w)$ 表示词串 $w$ 在语料中出现的次数。

### 5.3 数据平滑

**问题**: 数据稀疏导致零概率问题

**平滑基本思想**: "劫富济贫"，使零概率增值，非零概率下调

| 平滑方法 | 公式 |
|----------|------|
| **加1法 (Laplace)** | $p(w_i | w_{i-1}) = \frac{1 + c(w_{i-1}w_i)}{|V| + \sum_w c(w_{i-1}w)}$ |
| **加k法** | $p(w_i | w_{i-1}) = \frac{k + c(w_{i-1}w_i)}{k|V| + \sum_w c(w_{i-1}w)}$ |
| **回退法 (Backoff)** | 当高阶n-gram数据不足时，回退到低阶模型 |
| **插值法 (Interpolation)** | 结合不同阶数的n-gram模型 |

### 5.4 应用示例

**音字转换问题：**
$$CString = \arg\max_{C} p(C) \times p(P|C) \approx \arg\max_{C} p(C)$$

**汉语分词问题：**
$$Seg = \arg\max_{S} p(S) \times p(Text|S) \approx \arg\max_{S} p(S)$$

## 第6章 隐马尔可夫模型 (HMM)

### 6.1 HMM基本概念

HMM是一个五元组 $\lambda = (N, M, \pi, A, B)$

| 参数 | 含义 |
|------|------|
| $N$ | 状态数 |
| $M$ | 输出符号数 |
| $\pi$ | 初始状态概率分布 |
| $A$ | 状态转移概率矩阵 |
| $B$ | 输出概率（发射概率）矩阵 |

### 6.2 HMM三个基本问题

| 问题 | 描述 | 算法 |
|------|------|------|
| **评估问题** | 给定模型，计算观察序列的概率 | 前向算法 / 后向算法 |
| **解码问题** | 求最优状态序列 | Viterbi算法 |
| **学习问题** | 参数估计 | Baum-Welch算法 |

### 6.3 前向算法

**递推公式：**
$$\alpha_t(j) = [\sum_{i=1}^{N} \alpha_{t-1}(i) a_{ij}] b_j(o_t)$$

**终止：**
$$P(O|\lambda) = \sum_{i=1}^{N} \alpha_T(i)$$

### 6.4 Viterbi算法

**递推公式：**
$$\delta_t(j) = \max_{1 \leq i \leq N} [\delta_{t-1}(i) a_{ij}] b_j(o_t)$$

$$\psi_t(j) = \arg\max_{1 \leq i \leq N} [\delta_{t-1}(i) a_{ij}]$$

### 6.5 Baum-Welch算法

**E步:** 计算期望值
**M步:** 重新估计参数

迭代直至收敛。

### 6.6 应用

- **词性标注**: $Q = \arg\max_Q p(Q|O, \lambda)$
- **汉语分词**: 结合HMM与分词任务

## 第7章 词法分析技术

### 7.1 英语形态分析

#### 基本任务
1. 单词识别
2. 形态还原（词形还原）

#### 形态还原规则

| 变化类型 | 规则 | 示例 |
|----------|------|------|
| 过去时 -ed | 去掉ed | worked → work |
| 现在分词 -ing | 去掉ing | developing → develop |
| 第三人称 -s | 去掉s | works → work |
| 比较级 -er | 去掉er | colder → cold |
| 名词复数 -s | 去掉s | books → book |

#### 不规则变化
- 需要建立不规则变化词表
- 如：choose → chose → chosen

### 7.2 汉语自动分词

#### 分词的重要性
- 汉语句子分析的基础
- 词频统计、词典编纂
- 文本特征提取

#### 主要问题

**1. 分词规范问题**
- 单字词与词素的界限
- 词与短语的界限

**2. 歧义切分字段**

| 歧义类型 | 定义 | 示例 |
|----------|------|------|
| **交集型歧义** | 字串AB既可切为A/B，也可切为/AB | "大学生" → "大学/生" 或 "大/学生" |
| **组合型歧义** | 字串AB可切为/A/B，也可切为/AB | "将来" → "将/来" 或 "将来" |

**3. 未登录词识别**
- 人名、地名、组织机构名
- 新词语

#### 分词方法

| 方法 | 原理 | 特点 |
|------|------|------|
| **正向最大匹配法 (FMM)** | 从左到右匹配最长词 | 简单快速 |
| **逆向最大匹配法 (BMM)** | 从右到左匹配最长词 | 与FMM互补 |
| **双向匹配法** | 结合FMM和BMM | 处理歧义 |
| **基于统计的方法** | 使用HMM/CRF等模型 | 需要训练语料 |

## 第8章 句法理论

### 8.1 短语结构语法

**核心思想**: 句子由层次化的短语结构组成

**句法范畴：**
- **词类范畴**: N（名词）、V（动词）、Adj（形容词）、Adv（副词）
- **短语范畴**: NP（名词短语）、VP（动词短语）、PP（介词短语）、S（句子）

### 8.2 X-阶标理论

所有短语都遵循统一的结构模式：

```
XP → Spec X'
X' → X Compl
```

### 8.3 移动与疑问句

**Wh-移动**: 疑问词从深层位置移动到句首

例如：What did you buy? ← You bought what.

## 第9章 句法分析

### 9.1 短语结构分析方法

#### 移进-归约算法

| 操作 | 说明 |
|------|------|
| **移进 (Shift)** | 将输入词移入栈 |
| **归约 (Reduce)** | 将匹配产生式右部的符号归约为左部 |

#### CKY算法 (Cocke-Younger-Kasami)

- 自底向上分析
- 使用动态规划
- 要求文法为乔姆斯基范式 (CNF)

### 9.2 概率上下文无关文法 (PCFG)

**核心思想**: 为CFG的每个产生式赋予概率

$$P(A \to \alpha)$$

**约束**: $\sum_{\alpha} P(A \to \alpha) = 1$

#### PCFG三个问题

| 问题 | 算法 |
|------|------|
| 计算句子概率 | 内向算法 (Inside Algorithm) |
| 求最优分析树 | 外向算法 (Outside Algorithm) |
| 参数估计 |内外向算法 (Inside-Outside Algorithm) |

#### PCFG的优缺点

| 优点 | 缺点 |
|------|------|
| 利用概率减少搜索空间 | 条件假设苛刻 |
| 对概率小的子树剪枝 | 忽略结构上下文 |
| 定量比较语法性能 | 无法处理某些歧义 |

### 9.3 句法分析评估 (PARSEVAL测度)

$$Precision = \frac{\text{分析正确的短语数}}{\text{分析得到的所有短语数}} \times 100\%$$

$$Recall = \frac{\text{分析正确的短语数}}{\text{标准树库中的短语数}} \times 100\%$$

$$F_1 = \frac{2 \times P \times R}{P + R} \times 100\%$$

### 9.4 浅层句法分析 (语块划分)

**概念**: 识别句子中结构相对简单的独立成分

**基本名词短语 (Base NP):**
- 简单、非嵌套的名词短语
- 不含有其他子短语

**表示方法：**
- 括号分隔法: `[NP The cat] [VP sat] [PP on [NP the mat]]`
- IOB标注法: `The/B-NP cat/I-NP sat/B-VP`

## 第10章 语义分析

### 10.1 语义理论简介

| 理论 | 核心观点 | 优缺点 |
|------|----------|--------|
| **过程语义** | 句子语义=接受该句后执行的程序 | 简单但缺乏语言独立性 |
| **词汇分解学派** | 词义基于有限语义基元 | 类似元素学说，但基元难定义 |
| **条件真理模型** | 句子语义=命题在所有模型中的真伪 | 对上下文无关语义有效 |
| **情景语义学** | 语义与使用场景相关 | 引入事件、时间变量 |
| **模态逻辑** | 用公理系统刻画自然语言现象 | 难以涵盖特殊情况 |

### 10.2 格语法 (Case Grammar)

**提出者**: Fillmore (1966)

**核心观点**: 句子底层需要的不是主语/宾语等表层语法关系，而是施事、受事、工具等深层语义关系。

#### 格的类型

| 格 | 英文 | 说明 | 示例 |
|----|------|------|------|
| 施事格 | Agentive | 动作的发生者 | The boy opened the door |
| 客体格 | Objective | 动作影响的事物 | He bought a book |
| 工具格 | Instrumental | 动作的工具 | He opened the door with a key |
| 承受格 | Dative | 动作影响的有生物 | He is tall |
| 使成格 | Factitive | 动作形成的客体 | John dreamed a dream |
| 方位格 | Locative | 动作的处所 | He is in the house |
| 受益格 | Benefactive | 动作为之服务的对象 | He sang for Mary |
| 源点格 | Source | 动作来源 | He bought from Mary |
| 终点格 | Goal | 动作终点 | I sold to Mary |
| 伴随格 | Comitative | 伴随者 | He sang with Mary |

#### 格语法分析步骤

1. 找出句子的主要动词
2. 查找动词的格框架
3. 识别必备格和可选格
4. 填充格槽

### 10.3 语义网络

**定义**: 由概念和语义关系组成的有向图

**主要关系：**

| 关系 | 含义 | 示例 |
|------|------|------|
| **IS-A** | 是一种（继承关系） | 鱼 IS-A 动物 |
| **PART-OF** | 是一部分（整体-构件） | 桌面 PART-OF 桌子 |
| **IS** | 属性关系 | 中国 IS 首都-北京 |

**继承机制**: 子节点继承父节点的所有属性

### 10.4 语义角色标注

识别句子中各成分的语义角色（施事、受事、工具等）

## 第11章 机器翻译

### 11.1 基本翻译方法

#### 基于规则的翻译方法

```
源语言分析 → 结构转换 → 目标语言生成
```

**优点**: 保持原文结构，对规范句子效果好
**弱点**: 规则编写工作量大，一致性难保障

#### 基于中间语言的翻译方法

```
源语言 → 中间语言 → 目标语言
```

**优点**: 适合多语言互译
**弱点**: 中间语言设计困难

#### 基于实例的翻译方法 (EBMT)

**提出者**: 长尾真 (1984)

```
待翻译句子 → 相似实例检索 → 译文重组 → 目标语言
```

**优点**: 不要求源语言句子符合语法
**弱点**: 相似性难以把握，检索效率低

### 11.2 统计机器翻译

#### 噪声信道模型

$$T = \arg\max_T P(T|S) = \arg\max_T P(T) \times P(S|T)$$

| 组件 | 作用 |
|------|------|
| **语言模型 P(T)** | 保证译文流畅 |
| **翻译模型 P(S\|T)** | 保证翻译准确 |
| **解码器** | 搜索最优翻译 |

#### IBM模型

**五个模型 (Model 1-5):**

| 模型 | 特点 |
|------|------|
| Model 1 | 只考虑词汇翻译概率 |
| Model 2 | 增加对位模型 |
| Model 3 | 考虑 fertility（生育率） |
| Model 4 | 考虑位移模型 |
| Model 5 | 修正参数 |

#### 词汇对齐

给定句对 $(S, T)$，找出源语言词与目标语言词的对应关系

### 11.3 神经机器翻译 (NMT)

#### 编码器-解码器框架

```
源语言句子 → 编码器 → 上下文向量 → 解码器 → 目标语言句子
```

#### 注意力机制 (Attention)

$$c_i = \sum_{j=1}^{m} \alpha_{ij} h_j$$

$$\alpha_{ij} = \frac{\exp(e_{ij})}{\sum_{k=1}^{m} \exp(e_{ik})}$$

#### 主要架构

| 架构 | 特点 |
|------|------|
| **Seq2Seq** | 基础编码器-解码器 |
| **注意力机制** | 动态对齐，解决长距离依赖 |
| **Transformer** | 自注意力，完全并行化 |
| **BERT/GPT** | 预训练语言模型 |
